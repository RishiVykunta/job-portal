# Job Portal Web Application

Full-stack job portal application built with React, Node.js, Express, and PostgreSQL.

## Features

- Candidate: Search jobs, apply, upload resume
- Recruiter: Post jobs, view applications
- Admin: Manage users, jobs, applications, analytics

## Tech Stack

- Frontend: React, React Router, CSS
- Backend: Node.js, Express, PostgreSQL, JWT
- Authentication: JWT tokens
- File Upload: Multer for resume uploads

## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```
